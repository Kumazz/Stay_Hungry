# 模块
&emsp;&emsp;Python 模块是一个 Python 文件，以 .py 结尾，包含了 Python 对象定义和 Python 语句
&emsp;&emsp;模块能定义函数、类和变量，模块里也能包含可执行的代码

### 导入方式
*  import


```python
    # 导入模块
    import 模块名
    import 模块名1,模块名2...                 # PEP8 不推荐该写法
    
    # 调用功能
    模块名.功能名()
    --------------------------------------------------------
    import math
    print(math.sqrt(9))
