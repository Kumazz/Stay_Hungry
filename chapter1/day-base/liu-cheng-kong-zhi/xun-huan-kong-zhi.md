# 循环控制
### while 语法


```python
    while 条件: 
        条件成立重复执行的代码1
        条件成立重复执行的代码2
        ........
        
    # 计数器的习惯，在计算机中计数器一般默认初始值从 0 开始，这是一种工作习惯

    i = 0
    while i < 3:
        print('hello python')
        i += 1
    -----------------------------
    >>> hello python
    >>> hello python
    >>> hello python
      
```
