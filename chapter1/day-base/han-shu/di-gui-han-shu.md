# 函数加强
### 递归函数
&emsp;&emsp;变量作用域指的是**变量生效的范围**，主要分为两类: **局部变量**和**全局变量**


### 局部变量
&emsp;&emsp;定义在**函数体内部的变量**，即只在**函数体内部生效**

```python
    def testA():
        a = 100                 # 函数体内部定义的变量
        print(a)
    testA()                      >>> 100
    print(a)                     >>> name 'a' is not defined
        

```
&emsp;&emsp; 局部变量作用: 在函数体内部，临时保存数据，即当函数调用完成后，则销毁局部变量
