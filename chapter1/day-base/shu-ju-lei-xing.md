# 数据类型
### 认识数据类型
&emsp;&emsp; 应对不同的业务需求，把数据分成了 7 个不同类型，使用内置函数 type() 检查数据类型，代码如下:


```python
    myName = 'tom'
    print(type(myName))
    
    # 输出 myName 的数据类型
    <class 'int'>
```


*  数值: int（整型）、float（浮点型）
*  布尔型: True（真）、False（假）
*  字符串: str
*  列表: list
*  元组: tuple
*  集合: set
*  字典: dict
