# 开始写 Python
### 创建项目
&emsp;&emsp;打开 PyCharm -- [Create New Project] -- 选择项目根目录和解释器版本 -- [Create]，即可完成新建一个项目
![](/assets/QQ20200722-164126@2x.png)

### 创建文件
&emsp;&emsp;右键 -- [New] -- [Python File] -- 输入文件名 -- [OK]
![](/assets/QQ20200722-170505@2x.png)

### 运行文件
&emsp;&emsp;代码如下: 
```python
    print('Hello Python')

```
